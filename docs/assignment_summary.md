# Assignment Summary

This project follows the official mini-project brief: force a non-linear model to fail, diagnose the failure mode with controlled experiments, repair the cause rather than the symptom, and discuss threats to validity.

Core grading targets addressed:

- precise falsifiable research question;
- visible failure symptoms with subgroup diagnostics;
- causal hypothesis and controlled tests;
- non-trivial correction targeting shortcut learning;
- experimental validation with uncertainty;
- second failure mode on threshold dependence under class imbalance;
- mature threats to validity.
